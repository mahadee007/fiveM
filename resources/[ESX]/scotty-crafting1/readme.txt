**Feature**
- โต๊ะสร้างของระบบ UI เต็มรูปแบบ
- เมนูใช้งานง่าย
- แบ่งเป็นหมวดหมู่
- การปรับแต่ง
    - เพิ่มโต๊ะ
    - เพิ่มหมวด
    - เพิ่มไอเทม
    - เพิ่มจำนวนของที่ได้จากการคราฟ
    - เพิ่มจุดบอกโต๊ะในแมพได้ 
    - ปรับค่าเวลาการคราฟของไอเทมแต่ละชิ้น สามารถคูณตามชิ้นได้
    - ปรับอัตราล้มเหลวของไอเทมแต่ละชิ้น
    - ปรับแต่งภาษาได้
- ไม่ซ้ำซาก และ Unique ไม่มีแจกแน่นอน
- อัพเดตฟรีตลอดชีพ หากผู้ขายมีการอัพเดตเป็นตัวใหม่

**สคริปที่ต้องการในการทำงานร่วมกัน
- pNotify

**วิธีติดตั้ง**
1. โยนลง resources ของ fivem
2. เพิ่ม start scotty-crafting ลงใน server.cfg
3. ใส่ license key ที่ได้จากผู้ขายใส่ลงในไฟล์ config_sv.lua
4. ปรับแต่งของที่สามารถทำได้ในไฟล์ config.lua
5. เนื่องจากรูปไอเทมในแต่ละเซิฟจะไม่เหมือนกันกรุณาอย่าลืมใส่ไอคอนลงใน html/images/items
และเพิ่มที่อยู่ลงใน _resource.lua
